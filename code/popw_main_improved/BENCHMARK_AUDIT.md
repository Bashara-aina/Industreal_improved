**Key insight**: PTMA/MiniROAD's published cs/cv/csv mcAP numbers use **official PTMA splits that are not publicly available**. The local splits use assembly ID modulo logic and are NOT verified against PTMA's actual split definitions. The `act_mcAP` computed during default training is on the `cross_env` split вЂ” this cannot be directly compared to PTMA's published cs/cv/csv numbers. Use the `split_file` parameter for proper PTMA-style split evaluation.
